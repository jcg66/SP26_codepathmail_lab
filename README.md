# Android Lab 2 - *SP26_codepathmail_lab (CodepathMail)*

Submitted by: **James Girgis**

**CodepathMail** is a simple clone of the very popular [Gmail app](https://play.google.com/store/apps/details?id=com.google.android.gm&hl=en_US&gl=US).

Time spent: **1.5** hours spent in total

## Required Features

The following **required** functionality is completed:

- [x] **User can scroll through a list of 10 emails**

The following **optional** features are implemented:

- [x] User can press a 'Load More' button to see the next 5 emails displayed
- [ ] Each email displays additional information
